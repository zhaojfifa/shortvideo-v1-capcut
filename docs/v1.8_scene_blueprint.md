# v1.8 Scene Pack 工程蓝图（冻结版）

本文件用于归档 v1.8 的工程蓝图与边界，作为后续实现与评审的唯一依据。

## 版本定位
- 版本：v1.8
- 唯一切入点：Scene Pack（场景级素材资产）
- 目标：在不破坏 v1.7 的前提下，提升复用效率与规模化产能

## 继承与约束
- 继承：v1.7 pack 结构、资源策略、部署方式
- 不允许：新模型、自动剪辑、多账号调度、数字人/换脸
- 允许：Scene 的定义与拆分、Scene Pack 的结构、复用指标（效率/命中率）

## 核心定义
### Scene（最小完备单元 MSC）
一个 Scene 必须包含：
- video.mp4
- audio.wav
- subs.srt
- scene.json（见 schema：docs/schema/scene.schema.json）

### Scene Pack
- 目录结构：scene_packs/<scene_pack_id>/
- Manifest：scene_pack_manifest.json（见 schema：docs/schema/scene_pack_manifest.schema.json）

## 不破坏性原则
- v1.7 Pack 仍可独立生成与使用
- Scene Pack 为可选资产层，不替换 v1.7 主链路

## 冻结声明
从此刻起，任何 v1.8 讨论与实现必须能落位本蓝图章节；无法落位的需求视为超出 v1.8 范围。
