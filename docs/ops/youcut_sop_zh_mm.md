# YouCut 导入剪辑包 SOP（中缅双语） / YouCut SOP (中文 + မြန်မာ)

> 适用范围：手机端 YouCut 剪辑；从 /tasks/{task_id} 工作台获取 pack.zip / scenes.zip 后完成导出。
> Scope: Mobile YouCut editing; use pack.zip / scenes.zip downloaded from /tasks/{task_id} workbench.

---

## 1) 核心现实约束 / အရေးကြီးသော ကန့်သတ်ချက်
**YouCut 通常不能可靠导入 SRT 作为时间轴字幕**（不同版本差异大）。
因此本 SOP 默认采用“运营可执行”的通用方案：**使用纯文本字幕逐段复制粘贴**。

**YouCut သည် SRT စာတန်းထိုးကို အချိန်လိုက် timeline အဖြစ် ယုံကြည်စိတ်ချစွာ မထည့်သွင်းနိုင်တတ်ပါ** (version မတူကွဲပြားမှုများရှိ)။
ထို့ကြောင့် ဤ SOP တွင် **အများဆုံးအသုံးချနိုင်သောနည်းလမ်း** အဖြစ် **စာတန်းထိုး (txt) ကို အပိုင်းလိုက် ကူးယူ/ကပ်貼** သည့်နည်းကို မူလနည်းလမ်းအဖြစ်သတ်မှတ်ထားသည်။

---

## 2) 下载与解压 / ဒေါင်းလုပ် နှင့် ဖြုတ်ထုတ်ခြင်း

### 2.1 下载 / ဒေါင်းလုပ်
在 /tasks/{task_id} 工作台下载：
- pack.zip
- scenes.zip（若需要按场景作为素材池剪辑）

/tasks/{task_id} မှ အောက်ပါဖိုင်များကို ဒေါင်းလုပ်လုပ်ပါ။
- pack.zip
- scenes.zip (scene အလိုက် အသုံးပြုလိုပါက)

### 2.2 解压结构 / ဖိုင်ဖွဲ့စည်းပုံ
解压后你会看到大致结构（以实际为准）：
- raw/ 或 video.mp4（原视频）
- audio_mm/ 或 audio.wav（缅语配音）
- subs_mm/ 或 subs.srt（缅语字幕 SRT）
- subs_mm.txt（纯文本字幕，默认方案）
- scenes/（若解压 scenes.zip，会有 scenes/scene_001/...）

ဖြုတ်ထုတ်ပြီးနောက် ပုံမှန်အားဖြင့် အောက်ပါဖိုင်များကို တွေ့ရနိုင်သည် (အမှန်တကယ်ဖွဲ့စည်းပုံအပေါ် မူတည်)။
- raw/ သို့ video.mp4 (မူရင်းဗီဒီယို)
- audio_mm/ သို့ audio.wav (မြန်မာအသံသွင်း)
- subs_mm/ သို့ subs.srt (SRT စာတန်းထိုး)
- subs_mm.txt (စာတန်းထိုးစာသား txt — မူလအသုံးပြုနည်း)
- scenes/ (scenes.zip ကိုဖြုတ်ထုတ်လျှင် scenes/scene_001/... ရှိနိုင်သည်)

---

## 3) YouCut 新建项目与素材导入 / YouCut တွင် Project အသစ်ဖန်တီးခြင်း

### 3.1 新建项目 / Project အသစ်
- 打开 YouCut → 新建项目
- 选择导入视频（video.mp4 或 raw 中的视频）

YouCut ဖွင့်ပါ → Project အသစ်ဖန်တီးပါ → ဗီဒီယို (video.mp4 သို့ raw ထဲရှိဗီဒီယို) ကို ထည့်သွင်းပါ။

### 3.2 音频替换（关键步骤）/ အသံအစားထိုးခြင်း
- 将原视频音轨静音（Mute 原音）
- 添加配音音轨（audio_mm 或 audio.wav）
- 对齐起点：确保配音从 0:00 开始（必要时微调）

- မူရင်းအသံကို mute လုပ်ပါ
- audio_mm/audio.wav ကို ထည့်ပါ
- 0:00 မှ စတင်အောင် တိတိကျကျ ချိန်ညှိပါ

---

## 4) 字幕使用（两种方式）/ စာတန်းထိုးအသုံးပြုနည်း (နည်းလမ်း ၂ မျိုး)

### A) 少数版本可导入 SRT（可选）/ SRT ထည့်သွင်းနိုင်သည့် version (ရွေးချယ်နိုင်)
- 若 YouCut 支持导入字幕文件：尝试导入 subs.srt
- 导入后检查：时间轴是否对齐、字体是否正常、是否出现乱码

YouCut တွင် subtitle file ထည့်သွင်းနိုင်ပါက subs.srt ကို စမ်းသပ်ထည့်ပါ။
ထည့်ပြီးနောက် timeline တိကျမှု၊ font ပြဿနာ၊ စာလုံးမမှန်ခြင်း (乱码) ရှိ/မရှိ စစ်ဆေးပါ။

### B) 默认通用方式：用 subs_mm.txt 复制粘贴逐段添加（推荐）  
**这是默认且最稳的运营方式。**

1) 打开 subs_mm.txt
2) 按句子/段落复制一段字幕
3) 在 YouCut 里逐段添加文本字幕（Text）
4) 粗对齐时间：每段字幕覆盖对应口播片段即可（不追求毫秒级）

**မူလအသုံးပြုနည်း (အကြံပြု): subs_mm.txt ကို အပိုင်းလိုက် ကူးယူ/ကပ်貼**

1) subs_mm.txt ကို ဖွင့်ပါ
2) စာကြောင်း/အပိုင်းလိုက် ကူးယူပါ
3) YouCut ထဲတွင် Text subtitle အဖြစ် အပိုင်းလိုက် ထည့်ပါ
4) အချိန်ကို အကြမ်းဖျဉ်းတင်ညှိပါ (ms တိတိကျကျ မလို)

---

## 5) 导出规范（建议）/ Export အကြံပြုချက်
> 以“运营可执行 + 交付一致性”为目标。若你的团队已有固定参数，以团队标准为准。

建议：
- 分辨率：720p（或按渠道要求 1080p）
- 帧率：30fps
- 编码：H.264
- 音频：AAC
- 文件命名：YYYYMMDD_brand_product_taskid_voice.mp4

အကြံပြုချက်များ：
- Resolution: 720p (သို့ မီဒီယာလိုအပ်ချက်အရ 1080p)
- FPS: 30fps
- Codec: H.264
- Audio: AAC
- ဖိုင်အမည်: YYYYMMDD_brand_product_taskid_voice.mp4

---

## 6) 常见问题排查 / မကြာခဏကြုံရသော ပြဿနာများ

### 6.1 音画不同步 / အသံ-ရုပ် မကိုက်ညီ
- 通常是配音未从 0:00 对齐：把音轨起点拉回 0:00 再试
- 若仍漂移：检查是否对视频做了变速/剪切导致

အများအားဖြင့် audio ကို 0:00 မညှိထားခြင်းကြောင့် ဖြစ်သည်။ 0:00 သို့ ပြန်ညှိပါ။
Speed change / cut လုပ်ထားခြင်းရှိ/မရှိ စစ်ပါ။

### 6.2 字体显示异常/乱码 / Font ပြဿနာ
- 换字体或减小特殊符号
- 仍不行则改用更短的字幕片段

Font ပြောင်းပါ၊ သင်္ကေတများ လျှော့ပါ၊ အပိုင်းတိုတို ထည့်ပါ။

### 6.3 字幕与口播不一致 / စာတန်းထိုးနှင့် အသံမကိုက်
- 以口播为准：优先调整 subs_mm.txt 文案（运营可微调）
- 保持每段字幕尽量短（更易对齐）

အသံကို မူတည်ပြီး စာသားကို ချိန်ညှိပါ။ အပိုင်းတိုတိုထားပါ။
