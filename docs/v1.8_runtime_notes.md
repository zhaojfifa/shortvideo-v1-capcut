# v1.8 Runtime Notes (Scene Pack P0)

## Scope
v1.8 introduces Scene and Scene Pack as optional, reusable asset layers.
No changes are made to v1.7 pack structure or pipeline behavior.

## Key Capabilities
- Scene as Minimum Scene Cell (MSC)
- Scene Pack as reusable asset container
- Single Scene -> v1.7-compatible Pack
- Multiple Scenes -> one Pack without asset conflicts

## Non-Goals
- No new models
- No automatic scene detection
- No UI/workbench dependency
- No changes to v1.7 runtime path

## Compatibility
- v1.7 remains fully functional without Scene/Scene Pack
- Scene Pack usage is optional

## Status
v1.8 P0 completed and frozen.
