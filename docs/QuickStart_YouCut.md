# Smart Pack v1.7 (YouCut Ready)

This pack is a skeleton layout for importing assets into YouCut.

## YouCut import steps
1) Create a new project in YouCut.
2) Import `raw/raw.mp4` into the video track.
3) Import `audio/voice_my.wav` into the audio track.
4) Import `subs/my.srt` as subtitles.
5) Align audio with video and verify timing.
6) Preview and export as needed.

## Important
Do not rename files or folders.
Keep the directory structure exactly as provided.
